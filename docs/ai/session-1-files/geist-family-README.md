# Geist Font Family

The Geist font family by Vercel — a modern geometric typeface designed for developer tools and interfaces. Includes Sans, Mono, and experimental Pixel variants.

## Variants
- **Geist Sans** — Clean geometric sans-serif for UI text and headings
- **Geist Mono** — Monospaced variant for code, terminals, and data
- **Geist Pixel** — Experimental pixel/bitmap variants (Square, Grid, Line)

## Scale Ratio
1.2 (Minor Third) — xs(11px) → sm(13px) → base(16px) → md(19px) → lg(23px) → xl(28px) → 2xl(33px) → 3xl(40px)
