## How to Capture

Quick method using Chrome:
1. Open each URL
2. Cmd+Shift+4 (macOS) or use DevTools device toolbar for consistent viewport
3. Capture at 1440px wide for desktop reference
4. Save as: `01-factory-homepage.png`, `02-factory-cli.png`, etc.
5. Place in this directory
